# Getting started
## Installation
```bash
yarn
```
## Running the app
```bash
yarn run ios
```
## Running tests
```bash
yarn test
```
## Running lint
```bash
yarn lint
```
## Running e2e tests
First install maestro: [Maestro](https://maestro.mobile.dev/getting-started/installing-maestro)
Then run the following command:
```bash
maestro e2e/general.yaml
```
